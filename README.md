## 현재 시스템의 문제점
> 1. 면접 단계의 높은 부담감과 실전 연습 환경 부재
> - 특히 '강점 설명' 및 '직무 관련 전문성'을 묻는 질문에 대답하는 것을 가장 어려워하지만, 이에 대비할 수 있는 맞춤형 훈련 시스템 부재.
> 2. 면접 결과에 대한 개인화된 피드백 부재
> - 면접을 진행한 후에도 지원자에게 개인적인 피드백이 제공되지 않는 경우가 대부분. 이로 인해 지원자는 자신의 답변 중 무엇이 잘되었고, 구체적으로 어떤 점이 부족했는지 객관적으로 파악하고 다음 면접을 위해 개선할 수 있는 단서를 얻기 힘듦.


## 📌 제공하는 서비스 
| 구분               | 사전 준비 (서류 분석)                                             | 실전 면접 (Agent 인터뷰)                                                   | 결과 분석 (종합 피드백)                                                  |        
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------------ | ------------------------------------------------------------------ |
| **주요 기능**        | 이력서/자소서 업로드 및 텍스트 추출                                         | 사용자 응답 입력 및 AI 심화 질문 생성                                 | 면접 종료 후 인터뷰 대화 및 평가 결과 요약                                 |
| **핵심 기능**     | 서류 요약, 핵심 키워드 도출, 질문 전략수립         | 답변 평가(질문 연관성, 구체성), 인터뷰 진행 검토 | 각 전략 영역별 강점/약점 분석 및 종합 피드백 도출|
| **역할 정의**        | **면접 전략가**: 서류 기반 1:1 맞춤형 도입 질문 설계         | **실전 면접관**: 꼬리 질문 생성 및 답변 평가              | **커리어 코치**: 답변 스타일 분석 및 최종 보완점 제시              |
| **핵심 기술**   | LLM 기반 맞춤형 텍스트 분석 및 전략 템플릿 처리                 | Vector DB 기반 RAG 활용            | LLM 자동 평가 결과에 대한 자체 점검            |
